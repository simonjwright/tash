Download the current version of TASH

TASH is available for download in several formats, all of which contain the same set of files:

  • Unix tar file gzipped
  • Unix tar file compressed
  • Zip archive file

For Windows users who prefer not to (or cannot) build the libraries, there is a file for you to download which has Tcl, Tk, and Tash binary libraries ready for use with GNAT linker: tash831bwin.zip. This contains all the same files as the archive files above, with the addition of the Windows binary libraries.

TASH is developed on a 200 MHz PC (64MB RAM) running Windows 98 and GNAT 3.12p. It has also been tested with GNAT 3.12p in the following environments:

  • Sun/SPARC with Solaris 2.5.1
  • Red Hat Linux 6.2
  • Windows NT 4.0

How to get and install GNAT

You will need an Ada 95 compiler to build TASH. A description of GNAT may be found at Ada Home and it may be fetched via anonymous ftp from ftp://cs.nyu.edu/pub/gnat/ and its mirrors.

How to get and install Tcl/Tk

You will also need Tcl/Tk itself to build TASH. The home page is http://dev.scriptics.com/. This provides additional information about Tcl/Tk and instructions on how to get and install it. To install and build TASH, you will need, at a minimum, the binary installation and header files and libraries.
