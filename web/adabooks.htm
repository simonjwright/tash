Recommended Ada books

 

These books are presented here for your convenience in finding a suitable book for learning Ada. I have not read all of them but have either met and/or know the authors by reputation.

Ada 95 : Problem Solving and Program Design
by Michael B. Feldman, Elliot B. Koffman

This book is an excellent introduction to Ada for the novice programmer. It is used at several universities in their computer science program. It covers all the basics including abstract data types, generics, tasking, and object-oriented programming.

Ada As a Second Language
by Norman H. Cohen

Written for the experienced C programmer, this book can also be used as a reference (in addition to the Ada RM) for the experienced Ada programmer and designer.
Programming in Ada 95
by J. G. P. Barnes

This is not an introductory student textbook. Rather it is an extensive treatment of all aspects of Ada including object-oriented programming.
