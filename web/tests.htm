To build and run these test programs, cd to the tash820a/tests directory and execute make all test.

The TASHTest Program

The purpose of the tashtest program is to test the TASH Ada/Tcl binding. It does this by extending the Tcl interpreter with several Tcl commands implemented in Ada. It is intended that this test will eventually include tests for all of the thin binding. Currently, it tests a subset of the binding by implementing much of the sample code in chapters 30 through 32.2 of Ousterhout's book.

The Thick Binding Tests

The programs test_strings, test_integers, test_floats, test_lists, test_regexp, and test_format test the TASH thick binding packages implemented so far.
