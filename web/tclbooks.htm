Recommended Tcl/Tk books

 

These books are presented here for your convenience in finding a suitable book for learning Tcl/Tk. I have not read all of them so please judge for yourself.

Practical Programming in Tcl and Tk
by Brent B. Welch

You will truly learn practical programming in Tcl and Tk with this introductory, yet complete text. Better yet, Mr. Welch has recently updated the book to include some of the newest features of Tcl including threads, sockets, TCP/IP interfaces, secure shells, and web browser plug-ins.
Tcl and the Tk Toolkit
by John K. Ousterhout

Now a bit dated, this book was originally the only Tcl/Tk reference for many years. But, it is still a good introductory text for Tcl/Tk developers. Mr. Ousterhout is the original developer and lead of the current open source development effort of Tcl/Tk.
