Imagine creating platform-independent graphical user interfaces, performing regular expression pattern matching and using associative arrays as easily in Ada as you can in scripting languages such as Perl, Tcl, and awk. You can with TASH, a freely available Ada binding to Tcl/Tk.

Get started right away

To get started right away installing and using TASH, go directly to the Download page.

About TASH

TASH is a freely available Ada binding to Tcl/Tk. To read about TASH itself, see Introduction to TASH. See the Release Notes for information about what is new in this version.

About Tcl/Tk

Tcl/Tk is a great scripting language. It is easy to learn and is very portable. To learn more, go to the Tcl scripting pages at Scriptics, the leader in Tcl/Tk open source development, or visit the Tcl/Tk Consortium.

About Ada

Ada is a great systems programming language. It is a modern object-oriented, mission-critical language. To learn about Ada, start with the Ada FAQ and Tutorial at Ada Home.
